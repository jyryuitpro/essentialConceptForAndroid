Intent
- 의도, 의사

- 3가지 종류
    - A야 B좀 해줘
    - B좀 해줘
    ===================================
    - A야 B좀 해줘 그리고 다하면 알려줘

- 데이터를 같이 전달할 수 있다 (해당 전화번호로 전화를 걸 때, 해당 전화번호(데이터)를 전달)

- 명시적 인텐트 (Explicit Intent)
    - 누구야 이것 좀 해줘~
    - 호출될 대상을 명시하는 경우

- 암시적 인텐트 (Implicit Intent)
    - 그냥 이것 좀 해줘~
    - 호출될 대상을 명시하지 않는 경우
    - 인텐트 필터 (Intent- filter)
        - 암시적 인텐트를 보낸 경우, 인텐트가 처리할 수 있는지 확인할 때 사용
    - system

- 인텐트 호출 대상 V
    - 앱내에서
        - 엑티비티 끼리 (화면전환)
    - 외부(안드로이드 OS, 시스템), 앱 끼리(권한이 필요한다)
    - 사진 앱
        - 우리 앱 -> 시스템 -> 사진첩
        - 우리 앱 전화 걸기 버튼 -> 시스템 -> 전화 걸기 앱

- 인텐트 V
    - 결과가 필요한 경우 (2가지)
    - 결과가 필요하지 않은 경우 (1가지)
